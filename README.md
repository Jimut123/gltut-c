# Autoworld

These tutorials are for the batch 2015-2017 of RKMVERI, OpenGL course written in C. Please read [LICENSE](LICENSE) for more details.


## Installation

```
$ apt-get install libtool
$ chmod 777 autogen.sh
$ ./autogen.sh
$ ./configure
$ make
$ cd src
```
